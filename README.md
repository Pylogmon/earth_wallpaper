# earth_wallpaper

实时获取地球照片作为壁纸(Linux Only)

## 基础功能

- [x] 自动获取地球照片
- [x] 设置地球显示大小
- [x] 定时更新壁纸

## 支持接口

- [x] 向日葵八号
- [x] 风云四号

## 支持桌面环境

- [x] KDE
- [ ] Deepin
